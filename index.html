Packing List Label Generator

Build boxes, add products, print 4"×6" thermal labels to apply to the outside of boxes.

Product Catalog

▼
Auto-detects columns named SKU, Name, Barcode, EANBarcode/EAN, and UPCBarcode/UPC (case-insensitive) — works with plain exports and with inventory-system exports that have many other columns, which are simply ignored. If both EAN and UPC are present, both are searchable. This import is local to your browser only — to share the same catalog with the whole team, add/update a catalog.csv file next to index.html in the hosted repo instead.

Packing List History

Every time you click "Complete & Print Labels," the packing list is saved here. Use "Reprint / Edit" to load one back into the builder — add, remove, or fix quantities, then print again.
